Console de Transmission de l'Investigateur (Studio de Rédaction)

Session authentifiée. 🔓 Ce studio génère le bloc JSON de l'enquête, à coller dans le fichier articles.json du dépôt GitHub (la publication finale reste protégée par votre compte GitHub). « Prévisualiser » affiche le rendu exact en local avant publication.

Bloc JSON prêt à diffuser

1. Cliquez sur « Copier le JSON ».
2. Sur GitHub, ouvrez articles.json → ✏️ Modifier.
3. Collez ce bloc juste après le [ d'ouverture, suivi d'une virgule → l'article devient le nouveau Breaking News, l'ancien bascule dans le fil de droite.
4. Si vous avez une image, déposez-la dans le dossier images/ du dépôt.
5. « Commit changes » → en ligne en ~1 minute. 📰
