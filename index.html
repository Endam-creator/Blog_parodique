Console de Transmission de l'Investigateur (Studio de Rédaction)

Ce studio ne publie rien directement : il génère le bloc JSON de l'enquête, à coller dans le fichier articles.json de votre dépôt GitHub. Utilisez « Prévisualiser » pour voir le rendu exact avant publication (l'aperçu reste local, visible uniquement par vous).

Bloc JSON prêt à diffuser

1. Cliquez sur « Copier le JSON ».
2. Sur GitHub, ouvrez articles.json → ✏️ Modifier.
3. Collez ce bloc juste après le [ d'ouverture, suivi d'une virgule (le nouvel article apparaîtra en premier).
4. Si vous avez une image, déposez-la dans le dossier images/ du dépôt.
5. « Commit changes » → l'enquête est en ligne en ~1 minute. 📰
