🗜️ Optimiseur d'images

Compresse et redimensionne les images de tes enquêtes AVANT de les mettre sur GitHub. Tes photos trop lourdes ralentissent le site — cet outil les allège de 5 à 10× sans perte visible. Tout se passe dans ton navigateur, rien n'est envoyé nulle part.

Format PNG conservé (sans perte). Le gain vient du redimensionnement : réduire la largeur allège fortement les grandes images.
📁 Clique ici ou glisse tes images
JPG / PNG — plusieurs à la fois possible
💡 Comment faire :
1. Glisse tes images d'enquête ici.
2. Clique « Télécharger » sur chaque version optimisée (elle garde son nom en .png).
3. Mets ces versions allégées dans le dossier images/ de ton repo GitHub (remplace les lourdes). Pas besoin de modifier articles.json puisque les noms .png sont conservés.

Recommandé : largeur 1200px. Le PNG reste sans perte de qualité ; c'est le redimensionnement qui allège tes grandes images (souvent −70 à −90%).
